Important FAQ

What is the Document Object Model?

The Document Object Model (DOM) is a programming interface for web documents. It represents the structure of an HTML or XML document as a tree of objects, enabling developers to access, modify, and manipulate the content, structure, and styles of a web page dynamically.

How do you select an element from the (DOM)?

To select an element from the DOM, you can use methods like getElementById() for selecting by ID, getElementsByClassName() for selecting elements by class, getElementsByTagName() for tag-based selection, or querySelector() and querySelectorAll() for more flexible CSS-style selectors. These methods allow you to dynamically access and manipulate elements to create interactive web pages.

What is event delegation in the context of the DOM and why is it usefull?

Event delegation is a technique in the DOM where you add an event listener to a parent element instead of individual child elements. The parent listens for events on its children using the event's bubbling phase, where events propagate from the target element up through its ancestors.

How do you multiple an elements attributes and style using the DOM?

To modify multiple attributes and styles of an element in the DOM, you can use the `setAttribute()` method or directly assign values to its properties for attributes like `src` or `id`. For styles, you can use the `style` property to set individual CSS properties, such as `element.style.color = 'red'`, or assign multiple styles at once using the `cssText` property. This allows for dynamic updates to the appearance and behavior of elements on a web page.
